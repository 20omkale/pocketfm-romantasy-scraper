# Pocket FM Romantasy Scraper

This project implements an AI-assisted workflow to extract structured data from the Amazon Kindle Bestseller page for the Paranormal Romance category.

## Features

- Selenium browser automation
- BeautifulSoup HTML parsing
- Parallel scraping using ThreadPoolExecutor
- Data cleaning using Pandas
- Exported structured dataset (CSV)

## Extracted Fields

- Rank
- Title
- Author
- Rating
- Reviews
- Price
- URL
- Description
- Publisher
- Publication Date

## Tools Used

Python  
Selenium  
BeautifulSoup  
Pandas
